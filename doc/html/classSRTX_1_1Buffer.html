void SRTX::Buffer::abort_read ( )
virtual

Force all pending read blocking calls to be released immediately.

This function is used to cause a read blocking call to be aborted. The read blocking call should return indicating that an error occurred.

Reimplemented in SRTX::Ring_buffer< T >, SRTX::Double_buffer< T >, and SRTX::Double_buffer< SRTX::Message< T > >.
