bool SRTX::Syncpoint::wait ( const units::Nanoseconds &  timeout = units::Nanoseconds(0))

Wait for release from the syncpoint or optionally timeout.

Parameters
timeoutAbsolute time of timeout in nanoseconds. If timeout is 0, it is ignored, and the call waits forever.
Returns
True if released or timed out. False on failure.
Note
The associated mutex must be locked prior to calling this routine and will be locked when this routine returns.
